# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-23

### Added

- Initial OpenCode port of N-bench
- Core commands: setup, plan, work, interview, sync
- Review commands: impl-review, plan-review
- Prime command for codebase assessment
- Ralph mode (experimental autonomous execution)
- nbenchctl CLI with full feature parity

### Notes

This is an experimental port tracking upstream [N-bench](https://github.com/Nairon-AI/n-bench).
